# ED452C Source Map

This file records which repositories informed ED452C documentation and discussion drafts. It is meant to keep the portfolio trail clean without copying private placement records into public-facing course work.

## Source-use rules

- Use direct quotes sparingly, if at all. Prefer de-identified summaries.
- Do not copy raw transcripts, student work, teacher-specific notes, or identifiable placement details into ED452C.
- When a discussion post uses a placement pattern, record the source family here and log the post in `docs/discussion-log.md`.
- Use public repositories as portfolio-safe bridges when possible; use private repositories for source verification and richer context.

Reusable patterns for ED452C:

- intermediate learners can handle complex digital tasks when setup friction, debugging, and persistence are explicitly coached;
- sign-in, menu navigation, device troubleshooting, Minecraft world entry, and MakeCode access can hide computational-thinking evidence;
